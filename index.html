Preguntas a responder de Lab 3

  1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    2. No tengo mucha experiencia con el uso de !important, pero considero que se debería tener cuidado de que no se use excesivamente.

  2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    3. Pienso que por cuestiones de diseño, por sus dimensiones, colores, que por lo general las imagenes de fondo no son tan agradables para el usuario.

  3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    4. Considero que para que sea adaptable a cualquier dispositivo es bueno usar el %, y a lo mejor para cosas mas especificas el px. Realmente pt casi no lo uso.

  4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    5. Porque los espacios y líneas en blanco representan espacio ocupado, por lo que cuando se reduce a una línea se puede reducir el tiempo de ejecución
